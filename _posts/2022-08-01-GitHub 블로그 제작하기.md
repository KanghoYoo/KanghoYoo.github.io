# 깃헙(GitHub) 블로그 제작하기

##  GitHub Blog의 장단점

**장점**

- 원하는 형식을 자유롭게 커스텀 할 수 있다
- 호스팅 비용이 없다
- GitHub를 통한 커뮤니티가 활성화 되어있다.

**단점**

- 진입장벽이 있다
- 마크다운 형식을 알아야 한다
