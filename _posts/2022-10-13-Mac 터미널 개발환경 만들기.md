### 3. zsh shell 설치

- `brew install zsh`
- Bash shell에 추가 기능과 편의성을 제공해주는 shell

#### 3-1. Syntax Highlighting 설치

- `brew install zsh-syntax-hightlighting`

#### 3-2. Auto Suggestion 설치

- `brew install zsh-autosuggestiones`
